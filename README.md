# HappyHouse란 ? 
부동산 매매 정보사이트

+ 시 / 군 / 구 를 선택하면 해당 지역의 아파트들을 확인 할 수 있어요
+ 구글맵을 이용한 정확한 위치 서비스
+ 해당 아파트가 궁금하다면 ?  클릭해서 상세정보를 확인할 수 있어요 !
+ 주변 상가정보도 확인할 수 있어요 
+ 검색키워드를 이용해 주변 맛집 등을 검색할 수 있어요
+ 아파트 이름으로 검색도 가능해요
